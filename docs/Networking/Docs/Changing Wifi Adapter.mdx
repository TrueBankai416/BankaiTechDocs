On Debian press `ALT+F2`, now paste the following and press Enter.
```bash
nm-connection-editor
```

Now highlight the WIFI network you want and click on the Gear Icon at the bottom left.
Now you can choose which WIFI Card you wish to use for that network by changing the `Device`
